# 🛠️ DevToys - Developer's Swiss Army Knife

**DevToys** is a free suite of development tools designed to help developers with everyday tasks. From JSON formatting to JWT decoding, Base64 encoding to Regex testing, DevToys puts all the utilities you need in one beautiful, fast application.

---

## ✨ Why DevToys?

- **All-in-One Solution** - Multiple tools in a single application
- **Offline Support** - Works without internet connection
- **Modern UI** - Clean, intuitive interface
- **High Performance** - Built with modern technologies
- **Free & Open Source** - No cost, community-driven development

---

## 🎯 Key Features

### 🔄 Converters
- **JSON ↔ YAML** - Convert between formats instantly
- **JSON ↔ CSV** - Transform data structures
- **Base64 Encoder/Decoder** - Encode and decode text/files
- **URL Encoder/Decoder** - Handle URL-safe encoding
- **HTML Encoder/Decoder** - Escape and unescape HTML

### 📝 Formatters
- **JSON Formatter** - Pretty-print and validate JSON
- **XML Formatter** - Clean up XML structure
- **SQL Formatter** - Format SQL queries nicely
- **CSS Formatter** - Organize CSS rules
- **JavaScript Formatter** - Clean up JS code

### 🔍 Validators & Testers
- **Regex Tester** - Test regular expressions with sample text
- **JWT Decoder** - Decode and validate JWT tokens
- **Hash Generator** - Create MD5, SHA1, SHA256 hashes
- **Cron Parser** - Understand cron expressions
- **Color Picker** - Convert between color formats
